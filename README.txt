GOAL:
To create a memory game as a part of Udacity Front end Nanodegree program

How to load the game:
1) Downoad the zip file
2) Open index.html with suitable browser

The goal of the game:
There are 8 pairs of cards randomly arranged. The job is to match all 8 pairs.

How to play the game:
1)First click on any card and the symbol in the card gets displayed.
2)On clicking the second card there are two scenarios:
 	First, if the card matches the first card, both cards get displayed now 		following the similar process 7 more pairs should be matched to win the 		game.
	Second, if the card does not match the second card, the card is displayed 		for a second and both cards are hidden again.

3)Javascript
